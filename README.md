# 项目构建
**克隆项目**
```
git clone git@github.com:qn9301/react-native-aijinbao.git
```
**进入项目目录**
```
cd aijinbao
```
**下载node_module**
```
npm install
```
**在虚拟机上跑**
```
react-native run-ios
```
